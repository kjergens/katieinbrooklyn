If you haven't yet, check out Part one and Part two of this series which introduced color wheels and covered color palettes for the web.

In this tutorial we'll make a 2-color or 3-color palette using a print-optimized color wheel. The steps are the same as for web-optimized wheel.

1. Pick a Base Color

As before, choose a color you like to get started. I'm choosing red-orange, second from the top red.

2. Apply a Color Scheme

Choose one of four color schemes: complementary, triadic, split complementary, analogous.

Complementary

The complementary scheme uses colors 180 degrees apart. It's a very popular color scheme for logos, especially for companies that have been around awhile.
